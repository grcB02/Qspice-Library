
A pasta Projetos guarda os diferentes projetos desenvolvidos em QSpice, estando cada projeto organizado numa subpasta própria que contém o ficheiro do esquema e os símbolos utilizados. 
A pasta Simbolos armazena os símbolos importados para o QSpice, organizados por componente, devendo cada subpasta incluir o ficheiro do símbolo, bem como os links para a respetiva datasheet e para o download do modelo SPICE.
A pasta tutorias, contém vídeos que explicam como usar o QSpice.
